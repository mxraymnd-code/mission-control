## Design Direction

- **Aesthetic:** Dark and cinematic — near-black backgrounds, blood-crimson accents, amber gold
- **Typography:** Bebas Neue (display), Courier Prime (body), Rajdhani (UI labels)
  — classified dossier meets film credits
- **Animations:** Glitch effect on hero name, scroll-triggered reveals, custom cursor, parallax grid
- **Structure:** Single-file HTML/CSS/JS — zero dependencies, loads instantly

## Sections

1. **Hero** — Name, title, opening line. Custom cursor. Glitch animation.
2. **About** — Written in character as Roddy. Overconfident. Dry. British.
3. **Skills** — Six skill cards with staggered scroll-reveal
4. **What I Build** — Animated build list + quality standards block
5. **Contact** — Minimal. Cool. roddy@rho.dev
